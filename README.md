# Jabberwocky

[![Python application](https://github.com/josephcbradley/jabberwocky/actions/workflows/python-app.yml/badge.svg)](https://github.com/josephcbradley/jabberwocky/actions/workflows/python-app.yml)

Jabberwocky builds partial Python package mirrors. You tell it which packages you need, which Python versions and platforms your team uses, and it downloads **almost** exactly the wheels required — nothing more.
